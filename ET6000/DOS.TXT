(I) ET6000 DOS Utilities

  1. Vmode.com 
       Run this program to configure the ET 6000 display adapter 
       for the highest vertical refresh rates supported by your
       monitor. Higher refresh rate is less likely to cause eye 
       fatigue due to screen flicker. In addition, You can run 
       VMODE whenever you need to switch to a specific mode.

  2. Vdiag6.exe
       Run this program to test the capabilities and proper 
       functioning of your monitor for ET 6000 display adapter.


(II) Autodesk Device Interface 4.2  
    This driver provides support for Autodesk applications running 
    in DOS protected mode which require the Autodesk Device Interface 
    version 4.2 (ADI 4.2). Supported applications include AutoCAD 12 
    for DOS, AutoCAD 13 for DOS, and 3D Studio 4.0 for DOS.
